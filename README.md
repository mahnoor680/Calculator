# Calculator

A responsive and fully functional calculator built using HTML, CSS, and JavaScript. It performs basic arithmetic operations with a clean, modern, and user-friendly interface.

## Features

- Basic arithmetic operations (+, -, ×, ÷, %)
- Clear (C) button
- Backspace/Delete button
- Decimal number support
- Responsive design
- Real-time calculation
- Clean and intuitive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
